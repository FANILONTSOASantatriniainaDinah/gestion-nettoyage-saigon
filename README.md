# Planning Ménage ✦

Petite appli web toute simple pour gérer le tour de rôle du ménage à la maison. 🌸

## Ce qu'elle fait

- Affiche en grand qui s'occupe du ménage cette semaine
- Choix de la personne par semaine (boutons et liste déroulante)
- Indication de la disponibilité de chacune pendant sa semaine
- Case à cocher quand le ménage est fait
- Historique des semaines passées, exportable en JSON
- Les semaines s'ajoutent toutes seules, rien à mettre à jour

## Comment l'utiliser

Ouvrir la page, taper sur son prénom, et voilà. Tout est enregistré automatiquement dans le navigateur.

## Techno

Un seul fichier `index.html` : HTML, CSS et JavaScript, sans dépendance. Hébergé avec GitHub Pages.
